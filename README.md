# Static Resume (Eva Vasileska)

Open `index.html` in your browser to view the static resume. Files created:
- index.html
- css/styles.css
- assets/profile.svg

You can edit content and styles in the above files. For local testing, simply double-click `index.html` or serve with a static server.

Example (Python 3):
```
python -m http.server 8000
```
